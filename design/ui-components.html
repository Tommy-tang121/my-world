CSS Transform Formula

/* Container (rotates) */
transform: translateZ(-500px) rotateY({angle}deg);
transformStyle: preserve-3d;

/* Each card (positioned on ring) */
transform: rotateY({i * step}deg) translateZ(500px);
backfaceVisibility: hidden;
Container is translated back by radius, then rotated. Each card is rotated to its slot then pushed forward by radius.

4 Input Methods

MethodEventsBehavior
Mouse Dragmousedown / mousemove / mouseupDrag to rotate, 0.3°/px. Snap on release.
Scroll Wheelwheel (preventDefault)deltaY > 0 = next card, < 0 = prev
Touchtouchstart / touchmove / touchendSame as mouse drag, for mobile
Dots ClickclickSelect by index, shortest path rotation

Snap-to-Card Logic

// On drag/touch release:
const nearest = Math.round(-targetAngle / step);
targetAngle = -nearest * step;
const idx = ((nearest % count) + count) % count;
Rounds target angle to nearest step multiple, then derives card index.

Children Prop Pattern (Next.js)

// Works page (Server Component) passes cards as children:
<Gallery3D>
  {works.map(work => <WorkCard key={work.title} work={work} />)}
</Gallery3D>

SoftAurora Background

全屏 WebGL 流动背景：零依赖（浏览器原生 WebGL API，不使用 ogl 等第三方库），原生 WebGL + Perlin 噪声 shader，全屏大三角形覆盖，requestAnimationFrame 持续渲染。鼠标移动时背景产生轻微视差偏移。

基础信息

属性值
组件名SoftAurora
文件components/SoftAurora.jsx
类型Client Component（'use client'）
依赖零依赖，浏览器原生 WebGL API（不使用 ogl 等第三方库）
位置app/layout.jsx 中，fixed inset-0 z-0 pointer-events-none 全屏固定背景
层级关系SoftAurora 在最底层（z-0），上层内容用 relative z-10 浮在背景之上

接口（14 个 props）

Prop当前值说明
speed0.6流动速度
scale1.5噪声缩放
brightness1整体亮度
color1#f7f7f7主色 1
color2#e100ff主色 2
noiseFrequency2.5噪声频率
noiseAmplitude1噪声振幅
bandHeight0.5色带高度
bandSpread1色带扩散
octaveDecay0.1八度衰减
layerOffset0层偏移
colorSpeed1颜色变化速度
enableMouseInteractiontrue是否启用鼠标视差交互

实现原理

/* 渲染管线 */
原生 WebGL context → 编译 vertex + fragment shader
全屏大三角形覆盖（2 个三角形或 1 个超大三角形）
fragment shader 内实现 Perlin 噪声 → 多层八度叠加
requestAnimationFrame 持续更新 uniform（time + mouse）

/* 鼠标交互 */
mousemove → 更新 mouse uniform → shader 中偏移采样坐标
mouseInfluence=0.25 控制视差强度（轻微偏移，不喧宾夺主）

ClickSpark

全局点击火花动画：零依赖（Canvas 2D API），点击任意位置时从点击点向外发射 8 条白色线条，ease-out 缓动，400ms 消散。包住 Navbar + main，全局所有页面点击触发。

基础信息

属性值
组件名ClickSpark
文件components/ClickSpark.jsx
类型Client Component（'use client'）
依赖零依赖，Canvas 2D API
位置app/layout.jsx 中，包住 Navbar + main（全局所有页面点击触发）

接口（5 个 props）

Prop当前值说明

实现原理

/* 结构 */
<div className="relative"> /* 包 children */
  {children}
  <canvas className="absolute inset-0 pointer-events-none" /> /* 不阻挡点击 */
</div>

/* 动画流程 */
onClick(e) → 记录点击坐标 (x, y)
  for (i = 0; i < 8; i++):
    angle = i * (2π / 8)
    线条从 (x, y) 沿 angle 方向飞出 15px
requestAnimationFrame 持续渲染 → ease-out 缓动
线条向外飞同时长度逐渐缩短 → 400ms 后清除

8 条线条按 2π/8 = 45° 均分角度
